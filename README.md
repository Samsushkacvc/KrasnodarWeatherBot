# KrasnodarWeatherBot
Простой бот, показывающий текущую погоду в краснодаре в Краснодаре. Основан на aiogram и парсинге сайта с погодой
# Требования
- aiogram
- requests
- asyncio
- logging
- sys
- bs4
- bs
# Как запустить
1. В файле main.py вписываем свой токен
2. Запускаем файл start.bat
